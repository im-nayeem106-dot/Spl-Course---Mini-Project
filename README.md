# 🍔 Canteen Management System in C

A simple console-based **Canteen Management System** developed using the C programming language. This project is suitable for beginners and demonstrates the use of core C programming concepts.

---

## 📌 Features

- Display food menu
- Order food
- Automatic bill generation after ordering
- Calculate total bill
- User-friendly console interface

---

## 🛠 Concepts Used

- Functions
- Structures
- Arrays
- Loops
- If-Else Statements
- Recursion
- Global Variables

## 🚀 How to Run

### Using GCC

```bash
gcc canteen.c -o canteen
./canteen
```

### Using CodeBlocks

1. Open the project.
2. Copy the source code.
3. Build and Run (F9).

### Using Visual Studio Code

Compile:

```bash
gcc canteen.c -o canteen
```

Run:

```bash
canteen.exe
```
